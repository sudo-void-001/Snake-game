# 🐍 Snake Game in Python

A classic Snake Game built using Python's built-in `turtle` graphics library. Control the snake, eat food, grow longer, and try to beat your high score!

## 🎮 Features

- Classic snake gameplay
- Real-time score tracking
- High score system
- Random food spawning
- Snake body growth mechanics
- Collision detection:
  - Border collision
  - Self-collision
- Smooth movement controls using WASD keys
- Colorful UI built with Python Turtle

## 🎲 Game Rules

- Control the snake using **W, A, S, D**.
- Eat the red food to gain **10 points**.
- The snake grows each time it eats food.
- Hitting the wall resets the game.
- Colliding with your own body resets the game.
- Your highest score is preserved during the session.
